# Reusable booklet and component templates

`01_dol_2021plus-sbc-component-template.docx` is an editable federal Summary of
Benefits and Coverage component template. It provides reusable plan-comparison
tables and coverage-example structure, with a federal public-domain basis. It
is not a complete multi-benefit employer booklet and must be completed using
current federal instructions plus plan-specific facts.

The corpus still lacks a reusable full-booklet template whose license explicitly
permits adaptation. Completed employer guides do not satisfy that gap.
