# Prior-booklet scenario fixtures

These completed employer guides can act as a prior booklet only in a
scenario whose current employer identity matches the guide. `same_employer` is
a case relationship, not an intrinsic property of a public file.

| File | Intrinsic subtype | Scenario value |
| --- | --- | --- |
| `01_university-of-arizona_2026-benefits-guide.pdf` | Comprehensive public-university benefits guide | Broad prior-section and employer-program context |
| `02_city-of-houston_historical-new-hire-benefits-guide.pdf` | Historical 2018 municipal new-hire guide | Staleness and current/prior precedence |
| `03_intel_2026-new-mexico-annual-enrollment-guide.pdf` | Regional annual-enrollment decision guide | Short change-focused prior context |
| `04_seminole-county_2025_employee-benefits-guide.pdf` | Completed 2025 county guide | Genuine consecutive-year comparison with the retained 2026 Seminole County style-reference guide |

The former JPMC entry was moved to package-wide governing plan documents. The
Adobe guide was quarantined because every page is marked confidential. Original
provenance for all five audited downloads is preserved in
`source-docs/_provenance-catalogs/04_same-employer-prior-booklets.md`.

The 2025 and 2026 Seminole County files are different official annual guides,
not renamed copies. Their pair supports same-employer change detection across
costs, account limits, section order, glossary placement, and notices.
