# Broker-authored instructions

This leaf now contains three genuine broker/general-agency instruction sources:

- Word & Brown's California new-small-group implementation checklist;
- HUB International's 2023 open-enrollment compliance checklist; and
- Lawley's 2026 open-enrollment compliance checklist.

They add implementation and compliance-planning roles without pretending a
carrier form addressed to brokers was written by a broker. The copyrighted
checklists are test fixtures, not universal legal authority. The corpus still
needs controlled sold-case memos, override notes, and instruction emails.
