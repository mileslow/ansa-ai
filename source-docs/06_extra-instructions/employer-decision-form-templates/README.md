# Employer decision-form templates

These six public blank forms model how an employer may record plan selections,
renewal choices, contribution decisions, changes, and voluntary-benefit
elections. None is a completed client decision.

The set includes Members Health Plan NJ, Kaiser Oregon, Covered California,
Concordia, Health Net Oregon, and QualChoice forms. Tests must assert that blank
checkboxes and fields do not become facts. Completed customer uploads require
PII, banking, signature, and embedded-JavaScript controls where applicable.
