## Inclusion boundary

Washington HCA identifies exactly three PEBB vision plan options for 2026:
Davis Vision by MetLife, EyeMed, and MetLife Vision. Each carrier publishes a
separate governing certificate, so all three certificates are included. HCA's
three-page comparison is retained once under all-options because it compares the
same set side by side and expressly defers to the certificates if a conflict
exists. SEBB documents and the carriers' unrelated commercial plans are excluded.
