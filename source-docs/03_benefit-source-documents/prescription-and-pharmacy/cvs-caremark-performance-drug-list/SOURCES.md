# Prescription formularies

This bucket contains pharmacy support documents, not health-insurance plans.

`july-2026-formulary.pdf` is a July 2026 CVS
Caremark Performance Drug List. It tests therapeutic categories,
generic/preferred-brand distinctions, specialty-formulary references, and the
critical caveat that inclusion in a list does not guarantee plan coverage.
