# Prescription and pharmacy offering sources

This category follows the same offering-first layout as the insurance
categories. The documents have different operative roles and must not be
flattened into medical plan tiers.

| Offering folder | Document role |
| --- | --- |
| `kentucky-kehp-livingwell-ppo-rx/` | 2026 governing prescription-drug SPD for the LivingWell PPO option |
| `cvs-caremark-performance-drug-list/` | July 2026 formulary/support list; listing does not guarantee plan coverage |
| `new-jersey-shbp-sehbp-rx/` | July 2026 OptumRx-administered member and claims guide |

Each offering folder contains a `SOURCES.md` describing provenance, scope, and
classification. Formularies and member guides are supporting sources, not
standalone health-insurance plans.
