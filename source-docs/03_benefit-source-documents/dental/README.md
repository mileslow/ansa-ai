# Dental insurance offerings

The dental sources are organized by carrier/program offering, then by the actual
option represented by the authoritative document. A combined publisher brochure
stays combined when it governs multiple options; it is not copied into artificial
per-option files.

Retrieval and sibling-option audit date: **2026-07-17**.

Each offering folder contains a SOURCES.md with first-party URLs, the inclusion
boundary, document facts, checksums, and verification results.

The complete same-offering sibling sets are now present for the offerings that
have meaningful options. NYSHIP and Western Dental are single-option examples;
other NYSHIP bargaining-group documents and other CalHR dental carriers are
separate offerings, not missing tiers.
