# Dental companion schedules

`01_western-dental_2026-california-state-dhmo-schedule-of-benefits.pdf` is the
procedure-level Schedule of Benefits companion to the retained Western Dental
EOC. It provides ADA codes and member copayments that the EOC does not contain.
Treat the EOC and schedule as one multi-file plan example. Full provenance and
verification are in `docs/source-docs-valuable-benefit-additions.md`.
