# Package-wide governing plan documents

These sources govern several benefit sections and are too broad to label as a
single benefit or merely a prior booklet.

- `01_jpmorganchase_2025-combined-benefits-spd-plan-document.pdf` is a 399-page
  combined benefits guide, SPD, and plan document. Tests should segment it by
  section, respect governing-document precedence, and never transfer its facts
  to another employer.
