# Long-term disability sources

The current leaf contains four LTD-primary sources with different authority
levels:

| File | Source role |
| --- | --- |
| `01_university-of-michigan_basic-ltd-plan-booklet_2025.pdf` | Employer-paid, self-insured governing LTD plan booklet |
| `03_indiana-university-resident-med-plus_ltd-certificate-spd_2026.pdf` | Specialized resident-physician LTD certificate/SPD |
| `04_state-of-california_voluntary-ltd-benefits-guide_2026.pdf` | Cross-benefit enrollment guide covering LTD, accident, and critical illness |
| `05_university-of-alabama-system_standard-ltd-claim-faq_2026.pdf` | Claims-support FAQ, not coverage authority |

The Duke certificate was moved to `combined-disability/` because it governs
both STD and LTD. Detailed original provenance for all five scraped sources is
preserved in `source-docs/_provenance-catalogs/03_long-term-disability-original.md`.
