# Long-term disability sources

The current leaf contains six LTD-primary sources with different authority
levels:

| File | Source role |
| --- | --- |
| `university-of-michigan-basic-ltd/2025-plan-booklet.pdf` | Employer-paid, self-insured governing LTD plan booklet |
| `indiana-university-resident-med-plus-ltd/2026-certificate-spd.pdf` | Specialized resident-physician LTD certificate/SPD |
| `california-voluntary-ltd/2026-benefits-guide.pdf` | Cross-benefit enrollment guide covering LTD, accident, and critical illness |
| `university-of-alabama-standard-ltd/2026-claim-faq.pdf` | Claims-support FAQ, not coverage authority |
| `middlebury-college-unum-employer-paid-ltd/2026-certificate-spd.pdf` | Current employer-paid 60% LTD certificate/SPD |
| `university-of-missouri-metlife-core-buy-up-ltd/2026-core-and-buy-up-certificate-spd.pdf` | Current governing source comparing employer-paid Core and employee buy-up LTD options |

The Duke certificate was moved to `combined-disability/` because it governs
both STD and LTD. Detailed original provenance for all five scraped sources is
preserved in `source-docs/_provenance-catalogs/03_long-term-disability-original.md`.

The two 2026 governing-certificate additions are documented in
`docs/source-docs-valuable-benefit-additions.md`.
