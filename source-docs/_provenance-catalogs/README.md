# Preserved scrape provenance catalogs

These Markdown files are the original category catalogs for source groups that
were split after content-based auditing. They preserve URLs, retrieval dates,
hashes, and verification notes. Their historical directory labels are not the
current intrinsic classifications; use the live folder structure and audit
reports for current bucket semantics.

`duplicate-binaries/` preserves exact copies removed from the primary corpus so
the cleanup is recoverable. These files must not be loaded as evaluation cases.
