# Standalone legal and validation resources

- `01_cms_2026-part-d-creditable-coverage-and-lep-guidance.pdf` is current CMS
  guidance used to validate Part D disclosure content; it is not employer-ready
  notice copy.
- `02_dol_no-surprises-act-model-notice.docx` is an editable federal model
  notice with instructions and placeholders. It requires legal/currentness
  review and completion before participant use.

Full provenance, use constraints, hashes, and verification are recorded in
`docs/source-docs-valuable-booklet-instruction-additions.md`.
