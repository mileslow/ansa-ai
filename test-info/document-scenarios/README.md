# Booklet document scenarios

Generated by `npx tsx scripts/generate-booklet-document-scenarios.ts`.

These folders intentionally exercise missing information, extra source
documents, application-only intake, and rate mismatch behavior in the
`employee_booklet` builder and email generator. Each scenario uses distinct
reservoir plan documents when plan evidence is required.

- `00_complete-medical-dental`: Complete medical and dental package
- `01_hsa-selected-no-hsa-form`: HSA selected in application but no HSA form attached
- `02_extra-vision-plan-not-in-application`: Extra vision plan file not selected in application
- `03_application-only-progressive-intake`: Application only progressive intake
- `04_rate-sheet-plan-mismatch`: Application and spreadsheet plan mismatch
